Project Overview

A fully functional Stopwatch Web Application built using HTML, CSS, and JavaScript. It accurately measures and records time intervals with millisecond precision, supporting lap tracking and real-time display updates.

✨ Features
  • Start, Pause, and Resume timer with accurate millisecond precision
  • Lap recording — saves individual split times for each lap
  • Best lap highlighted in green, slowest lap in red automatically
  • +/− difference shown comparing each lap to the previous one
  • Animated SVG progress ring — fills every 60 seconds visually
  • Live status indicator — green pulse when running, yellow when paused
  • Reset clears all laps and returns timer to zero
  • Keyboard shortcuts — Space (Start/Pause), L (Lap), R (Reset)
  • Fully responsive — works on mobile and desktop screens
🎨 Design
  • Dark theme with purple and blue gradient color scheme
  • JetBrains Mono monospace font for timer display — clean and readable
  • Glowing buttons with hover and active animations
  • Smooth slide-in animation for each new lap entry
  • CSS variables used for consistent theming throughout
